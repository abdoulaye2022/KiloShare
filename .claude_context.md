# Contexte du Projet KiloShare

## Vue d'ensemble
Application mobile de partage d'espace bagages pour les voyageurs internationaux, principalement depuis/vers le Canada.

## Architecture
- **Frontend**: Flutter (iOS/Android)
- **Backend**: PHP Slim Framework
- **Database**: MySQL
- **Paiements**: Stripe Connect

## Workflow Git
- Branch principale : `master` (production)
- Branch de développement : `development`
- Branches Claude : `claude/[feature]-[date]`
- Toujours créer une nouvelle branche pour chaque feature
- Prefix des commits : `[Claude]`

## Commandes principales
- `new-feature [name]` : Créer une nouvelle feature
- `save-work` : Sauvegarder le travail
- `create-flutter-app` : Initialiser Flutter
- `create-backend` : Initialiser le backend

## Règles de développement
1. Ne jamais commiter directement sur master
2. Toujours travailler sur une branche claude/*
3. Commiter régulièrement avec le prefix [Claude]
4. Créer des petits commits atomiques
5. Documenter chaque nouvelle fonctionnalité
